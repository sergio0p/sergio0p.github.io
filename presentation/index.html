Practice Apps — ECON 101H & ECON 510

Why are textbooks so expensive? We choose them—but we don’t pay for them. What we’re really buying is the ancillaries: test banks, homework platforms, auto-graded exercises. That’s the moat that locks in $300 textbooks.

AI collapses the cost of building ancillaries. With AI, a single instructor can create practice apps that offer students unlimited, algorithmically generated problems with instant feedback—something no static textbook can match.

Cost Functions (101H)

  • Randomly generated cost table puzzles
  • 3 difficulty levels, Firebase leaderboard

Backward Induction (510)

  • Randomly generated extensive-form game trees
  • Animated resolution with SPNE path highlighted

Not All AI Are Created Equal

Chat AI (ChatGPT, Gemini, Claude.ai)

  • Copy-paste loop: describe → paste → breaks → repeat
  • AI only sees what you feed it
  • You are the debugger and the file system
  • Hard to customize or extend

Agentic AI (Claude Code, Cursor, Copilot)

  • Lives in your project—reads, writes, searches, and runs code directly
  • Breaks → sees the error → fixes it—without you lifting a finger
  • Teach it how you work—reusable skills and agents that learn your workflow

Same intelligence. The difference is agency.
