Universal identity for u₁*

In every σ(m,n) equilibrium with u₁* > 0, the smallest P1 atom x₁ faces only y₁ = 0 on its interior branch (truncation rule: ℓ ≤ k = 1 ⟹ ℓ = 1); all higher y_ℓ put P1 on the abandonment branch UT₁ = 0. Hence u₁* = u₁(x₁) reduces to a single interior term. Holds for any convex c.
u₁* − q₁ · [v₁ − c(w₂ − Δ − x₁)] + c(x₁) = 0
(★)
Quadratic specialization (c(e) = e², v₁ = v₂ = 1, Δ = 0): the P1 FOC at x₁ gives x₁ = q₁ / (1 + q₁), and (★) collapses to the QUICKSTART §2 closed form
u₁* − q₁² / (1 + q₁) = 0
(★q)

α-homotopy σ(2,2) → σ(1,2): tracking u₁*(α)

Conjugate of the upward construction in Docs/proof_ideas.md §168 (which adds a P2 atom for σ(n,n) → σ(n,n+1)). Here we remove P1's top atom x₂ — the m: 2 → 1 direction flagged in Docs/ranking_proof_audit.md §7(b).
Parameter. α ∈ [0, α*] is the mass on x₂ (so p = 1 − α is the mass on x₁). α = α* recovers full σ(2,2); α = 0 collapses to σ(1,2) (x₂ has no mass and drops from the support).
What is dropped along the path. P1's indifference between x₁ and x₂ holds only at α = α*. For α ∈ [0, α*), x₂ is a candidate atom carrying mass α whose FOC is enforced (local max of u₁) but whose payoff level is generically below u₁(x₁). The remaining 5 equilibrium conditions are kept for all α:
P1 FOC at x₁: q · c′(w₂ − Δ − x₁) − c′(x₁) = 0
(H.1)
P1 FOC at x₂: q · c′(w₂ − Δ − x₂) + (1 − q) · c′(w₂ − Δ − x₂ + y₂) − c′(x₂) = 0
(H.2)
P2 FOC at y₂: (1 − α) · c′(w₁ + Δ + x₁ − y₂) − c′(y₂) = 0
(H.3)
P1 indiff. at x₁ defines u₁*(α): q · [v₁ − c(w₂ − Δ − x₁)] − c(x₁) − u₁*(α) = 0
(H.4)
P2 indiff. at y₂: (1 − α) · [v₂ − c(w₁ + Δ + x₁ − y₂)] − c(y₂) = 0
(H.5)
5 equations, 5 unknowns x₁, x₂, y₂, q, u₁*(α) — for each fixed α.
Endpoint check at α = 0. (H.3) reduces to (1.2.2), (H.5) to (1.2.3), and (H.1) to (1.2.1) — exactly the σ(1,2) system; (H.4) then reads off u₁*(0) = u₁*(1,2). The "ghost" atom x₂ is still pinned by (H.2) but carries no mass.
Closing condition at α = α*. The dropped P1 indifference is restored:
Closure: q · [v₁ − c(w₂ − Δ − x₂)] + (1 − q) · [v₁ − c(w₂ − Δ − x₂ + y₂)] − c(x₂) − u₁*(α) = 0
(H.★)
At α = α*, equations (H.1)–(H.5) together with (H.★) reproduce the full σ(2,2) system (2.2.1)–(2.2.6). Define the gap
G(α) := u₁(x₂ | α) − u₁*(α)  =  LHS of (H.★) read as a function of α
(gap)
Then G(0) < 0 (x₂ off the σ(1,2) support sits strictly below u₁*) and G(α*) = 0. Monotonicity of u₁*(α) along the track gives the chain inequality u₁*(2,2) < u₁*(1,2) — see the analytical-results card below.

Analytical results along the track (any strictly convex c)

Below: IFT calculus on the 5×5 Jacobian J = ∂F/∂z with z = (x₁, x₂, y₂, q, u₁). All signs hold for arbitrary strictly convex c (c′ > 0, c″ > 0); quadratic plays no role. Shorthand: s₁ = w₂−x₁−Δ, s₂ = w₂−x₂−Δ, s₂′ = w₂−x₂+y₂−Δ, t₁ = w₁+x₁−y₂+Δ.
Result A — det(J) > 0 (failure mode #2: Jacobian non-degeneracy):
det(J) = (1−α) · [q·c″(s₂) + (1−q)·c″(s₂′) + c″(x₂)] · c′(s₁) · [c′(t₁)·c″(y₂) + c′(y₂)·c″(t₁)] > 0
(A)
Every factor is positive on-track by convexity + positive gaps. Closes failure mode #2: the IFT track never encounters a fold. Verified symbolically in Mathematica/jacobian.wl via FullSimplify[Det[Grad[sys,vars]]].
Result B — dq/dα < 0 (failure mode #1: q stays interior):
Cramer with the α-column ∂F/∂α = (0, 0, −c′(t₁), 0, −(v₂−c(t₁))) gives, after applying (H.3) and (H.5):
det(matrixQ) = [c″(x₁) + q·c″(s₁)] · [c″(x₂) + q·c″(s₂) + (1−q)·c″(s₂′)] · (v₂−c(t₁)) · [c″(y₂) + (1−α)·c″(t₁)] > 0,
hence dq/dα = −det(matrixQ) / det(J) < 0.
(B)
At the endpoints q(0) = 1/√2 ≈ 0.707 and q(α*) ≈ 0.365, both in (0,1). Strict monotonicity ⇒ q stays interior throughout [0, α*]. Closes failure mode #1.
Result C — du₁/dα < 0 ⇒ chain ranking u₁(σ(2,2)) < u₁(σ(1,2)):
Envelope at x₁ (kills the dx₁/dα term via (H.1)) yields du₁/dα = (v₁ − c(s₁)) · dq/dα.
Both factors signed: v₁ − c(s₁) > 0 (since q(v₁−c(s₁)) = u₁ + c(x₁) > 0) and dq/dα < 0 by (B).
Hence du₁/dα < 0 strictly on (0, α*).
(C)
Integrating: u₁(σ(2,2)) − u₁(σ(1,2)) = ∫₀^{α*} (du₁/dα) dα < 0. This is a cross-m ranking result, the kind §7(d) of ranking_proof_audit.md flagged as out of reach for the "envelope in q₁ alone" trick. Here the homotopy supplies the missing connecting path; the chain rule supplies the sign.
Failure mode #3 is subsumed by σ-uniqueness. If σ(1,2) and σ(2,2) are each unique in their class, then every zero of G(α) on the track corresponds to a σ(2,2) equilibrium ⇒ they all coincide ⇒ at most one zero. Existence of a zero follows from existence of σ(2,2). So under uniqueness, dG/dα need not be signed — the homotopy lands at the unique σ(2,2) by definition. Independently signing dG/dα along the track is straightforward except for one factor of the form v₁ − c(s₁) + c(s₂) − c(s₂′) whose sign is not obvious; see Mathematica/jacobian.wl for the FullSimplify output.
What stays open for the (2,2)→(1,2) case.
  • Verify Results A–C extend to arbitrary (v₁, v₂, Δ) — the proofs use no specialization, but we have only sweeps at v=1, Δ=0.
  • Track completeness: the IFT extension survives on [0, α*] as long as A and B hold; the boundedness of z(α) follows from FOC + convexity but a clean lemma would tighten the argument.
What stays open for the chain. The same machinery should run for any (n,n)→(n−1,n) or (n,n)→(n,n+1) step. The upward direction (n,n)→(n,n+1) is the version proof_ideas.md §168 sketches; doing the (m,n) generalization is the main remaining structural task.
